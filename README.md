# Installation
Télécharger le projet.  
Le décompresser.  
Ouvrir un terminal (Linux ou Windows) dans le dossier AngryBrids.  
Executer la commande :  
$ java -jar AngryBirds.jar  

Pour compiler soi même le projet :  
Ouvrir un terminal (Linux ou Windows) dans le dossier AngryBrids.  
lancer la commande :  
$ javac @Makefile
